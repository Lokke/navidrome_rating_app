# Navidrome Rating App


## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/lokke/navidrome_rating_app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd navidrome_rating_app
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
